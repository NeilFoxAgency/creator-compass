## Submission description draft

CreatorCompass turns a brand website into an evidence-backed direction for creator sponsorships. It safely reads a bounded set of public pages, builds a structured brand profile, scores ten readiness dimensions in deterministic code, ranks a reviewed creator-territory taxonomy, and uses GPT-5.6 as a final strategic adjudicator. The result is a shareable report with three Core territories, two Adjacent routes, one Experimental direction, two Risk zones, campaign concepts, a North Star test, evidence, uncertainty, and next steps. It helps brands answer “where do we belong?” before paying for a database or beginning outreach.
