# Security Policy

## Supported Versions

| Version | Supported          |
| ------- | ------------------ |
| 0.1.x   | :white_check_mark: |

## Reporting a Vulnerability

Please report security vulnerabilities to **neil@neilfoxagency.com**.

Include:
- Description of the issue
- Steps to reproduce
- Potential impact

We will acknowledge within 48 hours and aim to address critical issues promptly.

## Security Features

See [docs/security.md](docs/security.md) for details on SSRF protection, rate limiting, input validation, data handling, and more.

This project follows responsible disclosure. Do not publicly disclose vulnerabilities until fixed.
